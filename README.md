## 프로젝트 개요

* **프로젝트명:** 언리얼 Dedicated server with 스팀 연동
* **제작기간:** 약 2달 
* **플랫폼:** PC
* **언어:** C++
* **장르:** 언리얼 3D 전투 게임
* **사용 프로그램:** Unreal engine, Steam 플랫폼, Dedicated server

## 구현기능
- Unreal Dedicated 서버 연동
- 스팀 플랫폼 연동
- 실제 스팀 계정사용 가능
- 스팀 커뮤니티 서비스 창 사용 가능
